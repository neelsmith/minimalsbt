# Minimal sbt project

## File layout

The build file:  `build.sbt`, accompanied by `project/build.properties`

Plugins:  `project/plugins.sbt`

Source code:

-  `src/main/scala`
-  `src/test/scala`

Executable tutorial:

-  `src/main/tut`
-  `docs`.  NB:  can set github to serve `docs` directory as a web site.


## Key `sbt` tasks


-  `test`
-  `tut`
-  `publish`
